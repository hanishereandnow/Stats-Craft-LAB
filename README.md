# Stats Craft LAB

- [확률과 통계] 스터디를 진행한 Stats Craft LAB의 활동을 마치고, 학습내용을 정리한 깃허브입니다.
    - 정리본은 총 8가지 항목에 대한 내용으로, Wiki 탭에서 확인할 수 있습니다.
    - 코드 구현 정리는 총 5가지 항목에 대한 내용으로, Code 탭에서 확인할 수 있습니다.
- 모두의연구소의 LAB 활동으로, 모두의연구소 강남캠퍼스 오프라인 공간을 지원받아 진행되었습니다.
